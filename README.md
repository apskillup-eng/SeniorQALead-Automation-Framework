\# Senior QA Lead - Hybrid Test Automation Framework

A \*\*production-ready enterprise-grade\*\* C# hybrid automation framework demonstrating best practices in test automation architecture. This framework combines \*\*BDD (Reqnroll), Selenium WebDriver UI automation, and RestSharp API automation\*\* into a scalable, maintainable testing solution.

\## 🌟 Key Features

✅ \*\*BDD with Reqnroll\*\* - Gherkin syntax for business-readable test scenarios  

✅ \*\*UI Automation\*\* - Selenium WebDriver with Page Object Model pattern  

✅ \*\*API Automation\*\* - RestSharp v112+ (post-overhaul) for REST API testing  

✅ \*\*Multi-Browser Support\*\* - Chrome, Firefox with headless mode option  

✅ \*\*Configuration Management\*\* - Environment-specific configs with override support  

✅ \*\*Thread-Safe Design\*\* - ThreadLocal driver management for parallel execution  

✅ \*\*Fluent Assertions\*\* - Clean, readable assertion syntax  

✅ \*\*Comprehensive Documentation\*\* - Code comments and inline documentation  

✅ \*\*Scalable Architecture\*\* - Layered design following SOLID principles  

✅ \*\*CI/CD Ready\*\* - Pre-configured for GitHub Actions and Jenkins
